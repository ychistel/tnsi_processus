\subsubsection*{-3-~Afficher les processus}

La commande \textsf{ps} affiche les processus utilisateur liés à la console.

Avec certaines options, la commande renvoie des informations plus ciblées. Par exemple:

\begin{itemize}

\item l'option \textsf{-f} affiche plus d'informations sur les processus liés au terminal (console).\\

\item l'option \textsf{-u} suivi du nom utilisateur affiche les processus de l'utilisateur.

\item l'option \textsf{-e} affiche tous les processus de tous les utilisateurs.


\item l'option \textsf{-o} suivi d'arguments affiche les informations pour chaque processus et pour chaque argument passé en paramètres.

\end{itemize}

\begin{enumerate}
\item Afficher les processus utilisateur avec la commande \textsf{ps} puis avec la commande \textsf{ps -f}.

\item Afficher les processus utilisateur avec la commande \textsf{ps -u nom utilisateur}.

\item Reprendre la commande précédente en y ajoutant l'option \textsf{-f} pour avoir plus d'informations.


\item Afficher les processus utilisateur avec les argumentss \textsf{pid}, \textsf{ppid} et \textsf{command}.

\end{enumerate}

\subsubsection*{-4-~La gestion des processus}

\begin{enumerate}

\item Lancer un navigateur et observer dans les processus sa présence. 

\begin{enumerate}
\item Combien de processus enfants ont été créés ? \vspace{1cm}
\item Quel est le PID du processus parent.\vspace{1cm}
\end{enumerate}

\item Il est possible de terminer (tuer) un processus avec la commande \textsf{kill} suivi du numéro de processus.

Mettre fin en ligne de commande au processus associé à votre navigateur web. 

\item La console permet de lancer des programmes en tache de fond ou en arrière plan et d'avoir encore accès à la console. 

\begin{enumerate}
\item Écrivez un programme en Python qui boucle infiniment puis exécutez-le dans le terminal.

\item Est-il possible d'afficher les processus utilisateur ? Pourquoi ? Comment reprendre la main ? \vspace{1cm}

\item Pour éviter le problème précédent, on peut lancer le processus en arrière plan. 

Relancez votre programme Python en ajoutant \textsf{\&} en fin de commande.

\item Afficher les processus liés à la console. Afficher les processus avec leur durée d'exécution et vérifier que votre programme python s'exécute bien en arrière plan. 

\item Arrêter votre programme Python avec la commande \textsf{kill}.

\end{enumerate}

\item Python dispose du module \textsf{os} qui permet d'obtenir des informations sur le système d'exploitation.

Par exemple, il est possible de récupérer le \textsf{pid} du programme exécuté avec la méthode \textsf{getpid()}

\begin{enumerate}
\item Ajouter dans votre programme Python une variable \textsf{pid} qui a pour valeur le \textsf{pid} du programme Python en cours d'exécution. On veillera à afficher cette variable.

\item Lance votre programme en arrière plan et notez son PID.

\item Mettez fin à l'exécution de ce programme en utilisant son PID.
\end{enumerate}

\end{enumerate}
